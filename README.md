# MyReservationApp

## 概要 (Overview)
研究室の実験装置を管理・予約するためのカレンダーアプリケーションです。
従来の予約システムをベースにUIを大幅に改善し、Raspberry Pi（ラズパイ）環境での24時間稼働を前提として設計されています。

## 主な機能 (Features)
* **ドラッグ&ドロップ (Drag and Drop) 予約機能:** 直感的なマウス操作で予約時間の変更が可能。
* **予約の一括・重複自動調整コピー:** 複数選択（`Ctrl` / `Cmd` + クリック）による同時コピーと、時間重複時の自動調整。
* **UIの自動最適化:** カレンダーの列幅の自動拡張や、今日のセルへの自動スクロール機能。
* **利用頻度オートコンプリート (Autocomplete):** ユーザーの入力の手間を省く予測変換機能。
* **対応装置:** X線など、研究室の主要な測定装置の予約に対応。

## 技術スタック (Tech Stack)
* **バックエンド:** Java 17+ / Spring Boot / Gradle
* **フロントエンド:** Vanilla JS / HTML / CSS
* **データ管理:** ローカルファイルベース (`JSON` / `txt`)
